# Gym Tracker

## Mandatos del Proyecto

- **Gestor de Paquetes:** SE DEBE USAR `pnpm` exclusivamente. No usar `npm` ni `yarn` para evitar conflictos en el `pnpm-lock.yaml`.
- **Arquitectura:** 
  - `frontend/`: Aplicación Vue 3 (Vite).
  - `backend/`: Scripts de SQL para Supabase y lógica de servidor (si aplica).
- **Estilos:** Tailwind CSS. Las clases deben ser semánticas y reutilizables, centralizadas en `src/assets/main.css` usando `@apply`.
- **Diseño:** Mobile First prioritario.
- **Base de Datos:** Supabase (PostgreSQL).
- **Despliegue:** Vercel.

## Instalación

```bash
cd frontend
pnpm install
pnpm dev
```
